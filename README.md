## Pre-Requisite
-  Initialize your project directory
	> npm init
-	Install dependencies
	> npm install express cors prisma dotenv
-  Open prisma studio
	> npx prisma studio
- Synchronize model schema
	> npx prisma generate
- Running project
	> node .
